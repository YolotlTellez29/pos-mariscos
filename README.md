# Sistema POS para Restaurante de Mariscos

Este sistema fue desarrollado con FastAPI, MongoDB y un frontend HTML para gestionar pedidos en un restaurante de mariscos.

## Requisitos

- Python 3.12
- MongoDB (instalado en Windows)
- WSL2 con Ubuntu
- FastAPI
- Uvicorn
- email-validator
- bcrypt
- pydantic
